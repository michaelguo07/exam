Constrained sampling of phone numbers
from `sds315hw2_updated.Rmd` – Question 1
Building valid US phone numbers (XXX-XXX-XXXX) under multiple constraints using loops and sample().
random_phone_numbers <- character(10) for (i in 1:10) { first <- sample(2:9, 1) # 1st digit: 2–9 fourth <- sample(2:9, 1) # 4th digit: 2–9 # 2nd digit: cannot be 9 second <- sample(0:8, 1) # 3rd digit: cannot be 1 if 2nd is 1 if (second == 1) { third <- sample(c(0, 2:9), 1) } else { third <- sample(0:9, 1) } random_phone_numbers[i] <- paste0( first, second, third, "-", fourth, sample(0:9, 1), sample(0:9, 1), "-", sample(0:9, 1), sample(0:9, 1), sample(0:9, 1), sample(0:9, 1) ) } print(random_phone_numbers)

Key idea: translating verbal constraints on a sampling process into reproducible code.

Concept cheat‑sheet
Condensed concepts that recur in your homeworks and lectures.
Sample vs census
A census measures every unit in the population; a sample measures a subset. Your homework emphasizes pros/cons: censuses are complete but expensive, samples are cheaper but require careful design.
Units & variables
Each row represents a unit (e.g., chick at a time point). You classify variables as numerical vs categorical, discrete vs continuous, nominal vs ordinal.
Cross‑sectional vs longitudinal
Cross‑sectional: different units at a single time. Longitudinal: follow the same units over time (e.g., repeated measurements on the same chicks).
Experiments vs observational studies
Experiments involve random assignment to treatment; they let you make causal claims when well‑designed. Observational studies observe what already happens and rely on design tricks: matching, blocking, DiD, RDD, etc.
Confounding
A confounder influences both treatment and outcome (e.g., health consciousness in yoga vs stress). Good designs try to block, match, or randomize to mitigate confounding.
Natural experiments
Events like the Mariel Boatlift act as "as‑if random" shocks. You compare treated vs similar control cities to estimate causal effects when you can’t randomize.
Difference‑in‑Differences (DiD)
Compare changes over time between treated vs control groups. Critical assumption: parallel trends – absent treatment, both groups would have moved similarly.
Regression Discontinuity (RDD)
Treatment is assigned by a cutoff on a running variable (e.g., salary threshold). Assumes individuals can’t precisely manipulate their position near the cutoff.
Cohorts
Groups defined by a shared starting condition (e.g., disease‑free 20–30 year‑olds). You follow them over time to relate exposures to incidence.
Law of Large Numbers (LLN)
As sample size grows, sample averages tend to stabilize near the true mean; your simulations with increasing N illustrate this.
Sampling distribution
The distribution of an estimator (e.g., sample mean) over repeated samples. You built this by Monte Carlo simulation and saw it looks approximately Normal for large N.
Bootstrap
Resample with replacement from a single "true" sample to approximate the sampling distribution of an estimator and obtain standard errors and confidence intervals.
Population vs sample
The population is the full group you care about (e.g. all Texas chicken farmers); a sample is the subset you actually observe. Parameters describe populations, statistics describe samples.
Parameter vs statistic
A parameter is a fixed (usually unknown) number that summarizes a population (e.g. true mean weight); a statistic is a number computed from data (e.g. sample mean) that we use to estimate the parameter.
Codebook
A small table that lists each variable, its units, and a clear description. Your Homework 1 ChickWeight codebook is a good example: it forces you to think about what each column means before analyzing.
Descriptive vs analytical studies
Descriptive: summarize what a group looks like (e.g. average tip). Analytical: compare groups or relate variables (e.g. do coffee drinkers sleep less?) to answer "why" questions.
Prospective vs retrospective
Prospective: follow a cohort forward in time, recording exposures and outcomes as they happen. Retrospective: look back using existing records (e.g. past medical charts).
Blocking vs matching
Blocking is done before treatment assignment in an experiment (design phase) to balance key variables across groups. Matching is done after assignment (analysis phase) to pair treated and control units with similar characteristics.
Null vs alternative hypothesis
The null hypothesis (H₀) is the "no effect" or "no difference" story; the alternative (H₁) is what you’re trying to find evidence for (e.g. yoga reduces stress).
p‑value
The probability, assuming the null hypothesis is true, of getting a result as extreme or more extreme than what you observed. Small p‑values suggest the data would be surprising if the null were really true.
Standard error (SE)
The typical amount an estimator (like a sample mean) would vary from sample to sample. You estimated this both from formulas and via bootstrap in Homework 3.
Confidence interval
An interval built from data (e.g. estimate ± margin of error) designed so that a chosen percentage (often 95%) of such intervals would capture the true parameter in repeated sampling.
Type I vs Type II error
Type I: rejecting a true null (false positive). Type II: failing to reject a false null (false negative). Significance level controls Type I error; power is 1 − (Type II error).
